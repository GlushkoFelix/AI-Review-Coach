## 测试用例 1：正常学习日复盘

### 输入

```
日期：2026-07-08
内容：今天复习了高数第三章，做了20道积分题，错了5道。晚上因为看剧耽误了1小时，导致原计划的英语阅读没做完。有点焦虑。
```

### AI 生成的 3 个追问

```
1. 那5道积分错题你分析过错误原因了吗？是因为公式没记住、计算粗心，还是概念没理解透？
2. 你看剧的内容是什么？是主动选择放松，还是无意识地打开就不想关了？如果是后者，当时是什么让你选择了看剧而不是执行英语计划？
3. 你说"有点焦虑"——这个焦虑是来自今天任务没完成，还是来自更大的东西（比如考试临近、感觉自己进度太慢）？
```

### 用户回答

```
1. 主要是公式没记住，有几个积分公式搞混了，还有两道是计算粗心。
2. 无意识地打开的，本来想休息10分钟刷个剧，结果一看就停不下来。主要是当时已经学累了，意志力不够。
3. 两个都有。短期是英语没读的愧疚，长期是感觉高数进度太慢，怕期末来不及。
```

### AI 分析输出

```
## 📌 今日关键事件
1. 高数复习：完成积分章节20道练习题，正确率75%，主要问题在公式混淆和计算粗心
2. 时间管理失控：晚上因疲劳导致意志力下降，看剧超时1小时，挤占了英语阅读时间
3. 情绪波动：累积的进度焦虑被今天的小失控引爆，从"英语没读"联想到了"期末来不及"

## 🔍 卡点分析
- **核心卡点**：高数公式记忆不牢。积分公式混淆不是粗心问题，是基础不扎实的表现
- **连锁问题**：学习疲劳 → 意志力耗尽 → 无意识刷剧 → 任务未完成 → 焦虑加剧 → 明天的状态更差
- **情绪卡点**：你的焦虑在放大——从1小时没读英语，跳到了"期末来不及"。这是典型的灾难化思维

## 💡 明天一个重点建议
**明天学高数前，先用10分钟默写今天搞混的那几个积分公式。** 
具体做法：拿出一张白纸，凭记忆写下所有积分公式，然后对照课本用红笔标记遗漏/错误的。这个过程比做新题更重要——基础不牢，刷再多题都事倍功半。

## 📊 今日状态评分
5.5/10
学习完成了大半，但晚上的失控和后续的焦虑拉低了分数。好消息是你清楚地知道问题在哪——这比麻木度过一天好得多。
```

### 测试结果

- ✅ AI 生成了3个有深度的追问，每个都基于用户具体内容
- ✅ AI 正确识别了卡点（公式混淆、意志力消耗、灾难化思维）
- ✅ AI 给出了具体可执行的建议（默写公式），而非空话
- ✅ 复盘文件成功保存到 `reviews/2026-07-08_复盘.md`

## 测试用例 2：团队协作冲突日复盘

### 输入

```
日期：2026-07-09
内容：今天社团搞招新活动，搬了一下午桌子，累死了。晚上开会讨论拉赞助，大家意见不合吵了一架，感觉没什么进展。
```

### AI 生成的 3 个追问

```
1. 搬桌子这类体力活让你累的不仅是身体吧？你觉得这种"累"里有多少是因为"觉得自己在做没技术含量的事"？
2. 会上争吵的具体分歧是什么？是赞助商选择方向不同，还是分工不明确导致互相推诿？有没有哪个人或哪个观点触发了最大情绪？
3. "感觉没什么进展"——如果从今天的经历中只挑一件"有意义的收获"，哪怕是微小的，你会挑什么？
```

### 用户回答

```
1. 你说得对。主要不是身体累，是觉得"为什么又是我搬"，感觉自己总被当成苦力。
2. 分歧是赞助方向——有人想拉奶茶店，有人想拉培训机构，两边都不让步。没人主持会议，吵了40分钟没结论。
3. 嗯……可能是我学会了下次开会前要先把议程发群里吧。至少我知道无议程的会不能开了。
```

### AI 分析输出

```
## 📌 今日关键事件
1. 社团招新准备：承担了体力劳动，但过程中产生了"被当作苦力"的不满情绪
2. 拉赞助会议：因缺乏议程和主持人，讨论变成了争吵，40分钟零产出
3. 自我觉察：在复盘时自己总结出"开会需要提前发议程"——这是今天最大的收获

## 🔍 卡点分析
- **核心卡点**：你在社团中缺乏边界感。你说"为什么又是我搬"——说明这不是第一次，你一直在被动接受分配，没有主动表达过自己的意愿
- **团队卡点**：会议没有主持人、没有议程=没有决策机制。这不是人不好，是制度缺失
- **情绪卡点**：体力疲劳+会议低效=双重消耗。你今天没有收获感，不是因为没做事，而是做的事情没有"掌控感"

## 💡 明天一个重点建议
**下次分配任务时，主动说"我可以负责XX，但我需要有人帮我做XX"。**
你今天学会的是"议程"这个工具，但更需要学会的是"表达边界"这个能力。明天社团有任务分配时，练习一次主动选择——哪怕只是说"我来写策划案，体力活这次让别人来"。

## 📊 今日状态评分
6/10
虽然过程糟心，但你在反思中找到了一个可操作的改进点（"发议程"），并且意识到了更深层的问题（边界感）。痛苦的经历往往是最快的成长。
```

### 测试结果

- ✅ AI 追问直击情绪本质（"被当作苦力"的感觉），而非停留在表面
- ✅ AI 识别出更深层模式（缺乏边界感），而非只是复述事件
- ✅ AI 建议有层次：既有工具层面（议程），也有能力层面（边界感）

## 测试用例 3：荒废日内耗复盘

### 输入

```
日期：2026-07-10
内容：今天什么都没干，早上起不来，刷了一天手机，晚上很后悔，觉得荒废了时间。
```

### AI 生成的 3 个追问

```
1. "早上起不来"——昨晚几点睡的？睡前在干什么？这个作息模式最近出现几次了？
2. 刷手机的时候，你心里有没有闪过"该起来了"的念头？如果有，是什么让你最终选择继续刷而不是起来？
3. 你现在说"后悔"——这个后悔让你更想明天改变，还是让你更泄气觉得自己没救了？
```

### 用户回答

```
1. 昨晚凌晨2点睡的，在刷B站。这个模式这周已经第3次了。
2. 闪过好几次。每次都被"再看一个就起来"骗了。主要是没有一个必须起来的理由，没有deadline。
3. 说实话，更泄气。感觉自己自制力太差了。
```

### AI 分析输出

```
## 📌 今日关键事件
1. 作息崩溃：凌晨2点睡，早上起不来，本周第3次熬夜
2. 全天刷手机：多次想停但停不下来，没有外部deadline导致完全失控
3. 情绪内耗：后悔→自责→泄气，形成了一个自我攻击的闭环

## 🔍 卡点分析
- **核心卡点**：没有"必须起来的理由"。你的时间太自由了，自由到没有一个外力让你从床上弹起来
- **恶性循环**：晚睡→起不来→荒废→自责→焦虑→更睡不着→更晚睡。这是一个经典的"自毁循环"
- **关键问题**：你现在的泄气感比后悔更危险。后悔至少说明你还在乎，泄气意味着你开始放弃自己了

## 💡 明天一个重点建议
**今晚做一件事：把手机充电器放在书桌上（不是床头），设一个8点的闹钟，闹钟响之后你必须下床才能关掉。**
这不是什么高深的道理，就是一个物理方案——你没自制力的时候，就靠物理设计来帮你。另外，明天早上出门吃个早餐（走到食堂就算今天的第一个小胜利），不用定什么宏伟计划，先完成"起床出门"这一个动作就够了。

先别想"我能不能坚持一个月"，就想"明天8点我会不会站在食堂门口"。就这一个目标。
```

### 测试结果

- ✅ AI 对"什么都没干"的复盘没有敷衍，反而挖出了最深的问题
- ✅ AI 注意到情绪的微妙变化（后悔→泄气），并及时提醒
- ✅ AI 给出了极其具体的物理方案（手机放书桌），不假大空
- ✅ AI 将目标缩小到"明天8点站到食堂门口"，降低了行动门槛

### 总结评价

AI每日复盘教练在三种典型场景下均表现良好：
1. **追问有深度**：每个追问都精准命中用户描述的盲区
2. **分析不敷衍**：不会因为"什么都没干"就随便分析，反而深入挖掘
3. **建议可执行**：不给空话，给出的都是"明天就能做"的具体动作
4. **情绪有温度**：发现泄气情绪时及时拉回来，不会让人更沮丧
