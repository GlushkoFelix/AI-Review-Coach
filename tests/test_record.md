## 测试用例 1：正常学习日复盘

### 输入

```
2026-07-08 ：“今天复习了高数第三章，做了20道积分题，错了5道。晚上因为看剧耽误了1小时，导致原计划的英语阅读没做完。有点焦虑。”
```

### AI 生成的 3 个追问

```
1. 你算过那5道错题具体是哪些知识点失分吗？是积分技巧不熟还是对第三章的某个概念理解有漏洞——这能帮你判断今天的复习是否真的触及了核心弱项。
2. 看剧那1小时，你当时是纯粹想休息，还是在用刷剧逃避英语阅读的畏难情绪？如果换成15分钟正念或散步，会不会更高效地恢复专注？
3. 假如明天允许你“故意”先做半小时英语阅读，再奖励自己看一集剧，你觉得这种优先级调整是否能彻底消除今晚的焦虑感？
```

### 用户回答

```
1. 主要是分部积分和换元法的公式记混了，有几道是计算粗心。
2. 其实是逃避，看到英语阅读那么难就想去刷剧放松。
3. 应该会好很多，先完成最难的再有奖励。
```

### AI 分析输出（节选）

```
## 📌 今日关键事件
1. 完成了高数核心复习：系统复习了第三章，并进行了20道积分题的实战练习。
2. 发现了知识漏洞：定位到错误主要集中在分部积分和换元法公式混淆，以及计算粗心。
3. 因逃避而中断计划：晚上因畏难英语阅读而选择刷剧，导致原定计划未完成，并产生了焦虑情绪。

## 🔍 卡点分析
今天的核心障碍不在于时间不够，而在于情绪管理。
* 根因：你对“困难的英语阅读”产生了畏难情绪，这是一种典型的“逃避-焦虑”循环。
* 学习误区：复习中存在“粗心”或“记混”的浅层问题，今天的刷题可能更侧重“量”而非“质”。

## 💡 明天一个重点建议
从“最畏难”的部分开始，并将“奖励”设为任务完成后的结果。
具体行动：明天晚上，强制自己先做30分钟英语阅读。哪怕只读完一篇文章并查出生词，也是胜利。
完成这个“最硬骨头”后，明确告诉自己“现在可以看一集剧作为奖励”。

## 📊 今日状态评分
6/10分
理由：今天你成功发现了知识漏洞和情绪卡点，这是非常有价值的自我洞察。但最终执行结果打了折扣，且晚上陷入了负面情绪。
```

### 测试结果

- ✅ AI 生成了3个有深度的追问，每个都基于用户具体描述
- ✅ AI 正确识别了卡点（公式混淆、逃避-焦虑循环、学习重“量”轻“质”）
- ✅ AI 给出了具体可执行的建议（先做30分钟英语阅读，再奖励看剧）
- ✅ 复盘文件成功保存

## 测试用例 2：团队协作冲突日复盘

### 输入

```
今天社团搞招新活动，搬了一下午桌子，累死了。晚上开会讨论拉赞助，大家意见不合吵了一架，感觉没什么进展。
```

### AI 生成的 3 个追问

```
1. 搬一下午桌子这件事，是今天必须由你亲自完成的核心任务，还是可以优化的执行环节？如果重新分配体力，你会如何优先排序今天的精力投入？
2. 今晚会议上意见不合的焦点究竟是什么？是赞助方案本身的分歧，还是团队之间长期积累的沟通模式问题？
3. 如果明天要推动拉赞助这件事迈出一小步，你个人能主动发起的第一个具体行动（比如明天私下约一个持相反意见的成员单独聊5分钟）会是什么？
```

### 用户回答

```
1. 其实可以优化，不用我一个人搬一下午。应该提前分工，比如找人一起搬。
2. 焦点是赞助方案不同，有人要拉奶茶店，有人要拉培训机构，但根子上是没人主持会议。
3. 明天我私下分别找两个意见最大的人聊5分钟，了解他们真正的顾虑，再约个短会把议题定下来。
```

### AI 分析输出（节选）

```
## 📌 今日关键事件
1. 体力消耗在主次不分上：搬了一下午桌子，虽然累，但这并非你作为负责人的核心产出，属于可以优化和分配的低效执行。
2. 会议陷入僵局：晚上关于拉赞助的会议，因为方案分歧演变为争吵，没有记录和决策，没有主持和控场。
3. 发现团队协作的潜在问题：争吵表面是赞助方案选择，但根子是缺乏主持人、暴露出团队在议事规则和决策流程上的缺失。

## 🔍 卡点分析
今天最大的卡点是“临时搭配”和“议事机制缺失”。
* 精力分配：你习惯性地亲力亲为，陷入“低价值、可替代”的事务上，这说明缺少任务规划和分派意识，习惯自己一个人扛。
* 会议机制：意见不合本身不是问题，但没有主持人、没有明确的议程和决策规则，会议就变成了情绪的宣泄场，导致“吵半天，没结果”的挫败感。根因在于团队缺乏正式的议事规则，以及你作为核心成员，没有主动站出来承担引导角色。
* 情绪方面：你感觉“累”和“没什么进展”，背后是“深深的徒劳感和无力感”。你投入了体力，却看不到成果；你参与了争论，却发现无法推动事情。这种“忙而无功”的状态非常消耗人。

## 💡 明天一个重点建议
明天只做一件事：发起“15分钟+1对1”的私聊，不召开全体会议。
具体步骤：
1. 选对两个人：分别私下约今天晚上意见最尖锐的两个人（奶茶店与培训机构代表）各聊15分钟。
2. 明确目标：不是为了说服，而是为了真正搞清楚：对方的真正诉求是什么？有没有什么你忽略的需求？
3. 记录共识：把两个人说的“底线”和“可妥协点”记下来，找些共同点。比如：奶茶店想要“短期高人流”，培训机构想要“长期品牌转化”。
4. 以共识重启：如果私下聊出共识，另约一个短会（不超过30分钟），只讨论“我们在哪些点上可以共识”，而不是“哪个方案更好”。

这一动作的关键：把公开辩论，转换为私下摸底。把公开的“辩论赛”变成私下的“情报收集”。

## 📊 今日状态评分
6/10分
```

### 测试结果

- ✅ AI 追问直击会议机制问题（议程缺失），而非停留在情绪表面
- ✅ AI 识别出“低价值亲力亲为”和“议事规则缺失”两个模式
- ✅ AI 建议有可操作性：从“1对1私聊”到“短会重启共识”

## 测试用例 3：荒废日内耗复盘

### 输入

```
今天什么都没干，早上起不来，刷了一天手机，晚上很后悔，觉得荒废了时间。
```

### AI 生成的 3 个追问

```
1. 你在刷手机的过程中，有没有某个瞬间内心闪过“该停下了”的念头？如果当时有，是什么具体的念头或外部因素让你最终没有放下手机？
2. 早上起不来的那一刻，你最真实的内在感受是什么——是身体疲惫、对一天要做的事感到抗拒，还是觉得反正也没人催就放任了？这种感受指向了你平时忽略的哪个深层需求或恐惧？
3. 如果明天你只能完成一件很小、但能让你感到“这一天有分量”的事，那会是什么？如何在你睁开眼后的15分钟内就启动它，避免重蹈覆辙？
```

### 用户回答

```
1. 闪过好几次，但每次都想“再看5分钟”，然后就过了1小时。主要是没人催，自己管不住自己。
2. 其实是逃避，一想到今天要复习高数就烦，干脆躺着刷手机。深层可能是对困难任务的抗拒。
3. 明天先做一道高数题，就一道。醒来后15分钟内坐到书桌前，把手机放另一个房间。
```

### AI 分析输出（节选）

```
## 📌 今日关键事件
1. 目标崩塌：原定的高数复习计划，因早起失败而完全搁置。
2. 时间黑洞：陷入“刷手机-内疚-刷手机”的循环，外部无人打扰，内部缺乏自制力，让一整天从指缝中溜走。
3. 情绪反馈：晚间的“后悔”和“荒废感”是良知的信号，但也消耗了能量，导致无法为明天做有效规划。

## 🔍 卡点分析
今天的核心卡点在于“启动困难”和“即时满足”的对抗。
* 根因分析：早上起不来的那一刻，本质上是对困难任务的“预期性焦虑”。大脑为高数复习贴上了“痛苦”、“复杂”、“耗时长”的标签，所以启动了最简单的逃避机制——刷手机。而“再看5分钟”这个想法，是利用了人类被“非完成感”折磨的心理弱点，让你愿意用一个已知的即时奖励（多巴胺）去交换一个未知的长期回报（学习带来的成就感）。
* 情绪分析：你晚间的“后悔感”非常真实且痛苦，但这恰恰是你改变的起点。你今天的情绪状态是一个“后悔-放纵”的闭环：因为目标过难而逃避，因逃避而内疚，因内疚而更想用刷手机来缓解，最后导致更深的后悔。问题不在于你不够聪明或不够努力，而在于你让“高数复习”这个庞然大物，在早上七点就成功吓倒了还没来得及激活主控权的你。

## 💡 明天一个重点建议
明天醒来后，执行“开门杀”行动：
1. 起床后0-15分钟：不要碰手机，不刷牙不洗脸。直接走到书桌前，坐下，拿出高数题集。
2. 第15-30分钟：只做“一道题”。没错，就是一道。做完即可，哪怕用30分钟。
3. 做完后：立刻奖励自己一个明确的事，比如“现在可以去安心吃早饭、看15分钟手机”。

为什么有效？这个方法直接绕开了“要不要复习高数”这个巨大的决策负担，只问一个极简的问题：“要不要做一道题？” 这个行动小到你的大脑无法抗拒，“惯性”会带你自然地完成。完成这道题带来的微小成就感，就是打破“后悔”链条的第一锤。

## 📊 今日状态评分
3/10分
理由：今天状态极低，几乎完全陷入负面循环，自我效能感受到了打击。但给你6分的迟疑和3分的坚决在于：你清晰地意识到了问题，并且通过复盘准确地指出了“逃避”和“任务过大”这两个核心。这份清醒本身，就是明天反弹的宝贵基石。你离重新掌控自己，只差一个“一道题”的距离。
```

### 测试结果

- ✅ AI 对“什么都没干”的复盘没有敷衍，反而深入挖掘“预期性焦虑”
- ✅ AI 准确识别了“后悔-放纵”闭环和“即时满足”机制
- ✅ AI 给出了极简的物理方案（起床15分钟内坐下做一道题）

### 总结评价

AI每日复盘教练在三种典型场景下均表现良好：
1. **追问有深度**：每个追问都精准命中用户描述的盲区，且引导方向层层递进
2. **分析不敷衍**：不会因为“什么都没干”就随便分析，反而深入挖掘根因和情绪模式
3. **建议可执行**：不给空话，给出的都是“明天就能做”的具体动作（如“先做30分钟英语阅读”、“约两个人1对1私聊”、“起床后15分钟内做一道题”）
4. **情绪有温度**：发现泄气、焦虑情绪时，既指出问题也给出希望，避免让人更沮丧
